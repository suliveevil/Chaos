# Universal Link


[[Apple苹果]]通用链接，标准 [[URL]] 格式。

优点：
通用性
灵活性
安全性
隐私性

要求：
有一个注册的[[域名]]
通过 SSL 访问域名（HTTPS）
域名下配置 apple-app-site-association 文件
